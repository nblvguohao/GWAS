
# 真实作物数据集下载指南

## 🎯 推荐数据集（按易获取性排序）

### 1. 玉米数据 (Panzea) - 最推荐
```bash
# 访问 https://www.panzea.org/data
# 选择 "Genotypes" 和 "Phenotypes"
# 下载 HapMap3 基因型数据
# 下载对应的表型数据
```

### 2. 水稻数据 (Rice Diversity) - 推荐
```bash
# 访问 https://ricediversity.org/
# 注册账号
# 下载 RDP1 数据集
# 包含 300 个品种的完整数据
```

### 3. 拟南芥数据 (1001 Genomes) - 最简单
```bash
# 访问 https://1001genomes.org/
# 直接下载 VCF 文件
# 下载表型数据
```

## 🔧 数据处理建议

### 下载后处理步骤:
1. **质量控制**
   - 过滤低质量SNPs (MAF < 0.05)
   - 移除高缺失率样本/位点
   - 一致性检查

2. **格式转换**
   - VCF → CSV/PLINK格式
   - 表型数据标准化
   - 样本ID匹配

3. **数据验证**
   - 检查基因型-表型对应关系
   - 验证数据完整性
   - 统计分析

## 📝 申请真实数据需要的材料

如果需要申请访问权限，准备：
- 学术机构邮箱
- 研究计划说明
- 数据使用承诺书
- 合作协议（如需要）

## ⚠️ 重要提醒

1. **数据使用协议**: 仔细阅读每个数据库的使用条款
2. **引用要求**: 使用数据时需要正确引用原始研究
3. **数据共享**: 部分数据集可能要求分享研究成果
4. **更新频率**: 定期检查数据集更新
