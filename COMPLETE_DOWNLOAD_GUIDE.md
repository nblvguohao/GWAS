## 🔍 下载失败原因分析

### Rice3k Google Drive 下载失败
**主要原因**:
1. **文件数量限制**: 文件夹包含 >50 个文件，`gdown` 工具限制
2. **访问权限**: 某些文件可能需要特定权限
3. **网络限制**: 防火墙或代理阻止访问
4. **文件大小**: 总数据量可能很大

### 错误信息解读
```
The gdrive folder with url: https://drive.google.com/drive/folders/1H6XL9IHDvXR8Suq64bd1NxH_YghGdUYC
has more than 50 files, gdrive can't download more than this limit.
```

## 🔄 替代数据源

### 数据源 1: 3K Rice Genomes Project

#### 官方数据
- **网站**: https://snp-seek.irri.org/
- **数据量**: 3,000+ 水稻品种
- **数据类型**: 全基因组 SNP 数据
- **访问方式**: 需要注册账号

#### 下载步骤
1. 访问 https://snp-seek.irri.org/
2. 注册账号并登录
3. 选择 "Download" 或 "Export" 功能
4. 筛选所需品种和 SNP
5. 导出数据

### 数据源 2: Rice Diversity Project

#### 官方数据
- **网站**: http://www.ricediversity.org/data/
- **数据量**: 1,600+ 水稻品种
- **数据类型**: 基因型 + 表型数据
- **访问方式**: 直接下载

#### 关键数据集
- **HDRA_GWAS**: 高密度水稻关联分析数据
- **Phenotype_Data**: 多个环境表型数据

### 数据源 3: USDA GRIN Database

#### 官方数据
- **网站**: https://www.ars-grin.gov/
- **数据类型**: 美国水稻种质资源
- **特点**: 包含丰富的表型数据

## 🆘 故障排除

### 常见问题及解决方案

#### 问题 1: Google Drive 访问受限
**解决方案**:
- 使用 VPN 或代理
- 尝试不同时间段访问
- 联系有访问权限的同事

#### 问题 2: 文件下载不完整
**解决方案**:
- 使用下载管理器
- 检查文件大小是否匹配
- 重新下载损坏文件

#### 问题 3: 数据格式不识别
**解决方案**:
- 运行格式检测脚本
- 手动检查文件前几行
- 查看数据说明文档

#### 问题 4: 内存不足
**解决方案**:
- 分批处理大数据文件
- 使用高效的数据格式
- 增加虚拟内存
