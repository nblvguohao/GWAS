## 🧮 **Stacking模型构成**

### 🏗️ **基础模型组合**

我们的Stacking模型通常包含：
1. **GBLUP** - 线性基线
2. **MLP** - 非线性深度学习
3. **CNN** - 局部特征提取
4. **Ridge回归** - 元学习器

### 🔄 **集成策略**

```python
# Stacking架构
base_models = {
    'GBLUP': GBLUP(),
    'MLP': MLP(),
    'CNN': CNN(),
    'CNN-Huber': CNN(loss='huber')
}

# 元学习器
meta_learner = Ridge(alpha=1.0)

# 交叉验证训练
stacking_predictions = cross_validate_stack(base_models, meta_learner)
```

## 💡 **Stacking成功的关键因素**

### ✅ **多样性优势**

1. **算法多样性**: 线性(GBLUP) + 非线性(MLP/CNN)
2. **损失函数多样性**: MSE + Huber
3. **特征提取多样性**: 全连接 + 卷积
4. **正则化多样性**: L2 + Dropout

### 🎯 **互补性分析**

| 模型对 | 互补性 | 说明 |

| GBLUP + MLP | 高 | 线性 vs 非线性 |
| CNN + MLP | 中 | 局部 vs 全局特征 |
